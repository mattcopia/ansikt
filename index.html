Introduction · 01

ansikt

beauty

Ansikt — Norwegian for "face" — is a bespoke brow and facial treatment studio. Every visit is tailored, every space considered. We believe in natural beauty enhanced, not masked; in calm, warm spaces that feel like a pause from the world.

This brand identity system defines the visual language of Ansikt. It provides the colours, typography, components, and guidelines needed to maintain consistency across all touchpoints — from digital to print.

Introduction · 02

Logo

The Ansikt logo is a bold display serif wordmark with high stroke contrast, a decorative swash on the lowercase "a", and angular cuts on the "k" and "t".

ansikt
Logo set in Ritualist — SVG asset to be supplied
ansikt
Inverted variant on dark backgrounds

Usage Rules

DO
  • Maintain minimum clear space of 1× the height of the "a" on all sides
  • Minimum display size: 120px width for digital
  • Use on solid light or solid dark backgrounds only
  • Use the supplied SVG/PNG — do not recreate in a different font
DON'T
  • Stretch, skew, or rotate the logo
  • Change the logo colours (use only black, white, or brand charcoal/linen)
  • Add drop shadows, outlines, or other effects
  • Place on busy imagery without a solid overlay
  • Display below 120px width on digital, 25mm on print

Typography · 03

Usage rules

Clear guidance on when to use each font and what to avoid.

DO
  • Use Ritualist for anything the user reads first — headings, titles, hero text, nav
  • Use Plus Jakarta Sans for everything functional — body, buttons, nav, forms
  • Use Courier Prime for prices, durations, and data tables
  • Minimum 14px for any text on screen
  • Use letter-spacing ≥0.02em on uppercase labels
  • Use ink-1 for headings, ink-2 for body, ink-3 for muted/labels only
DON'T
  • Use Ritualist for body paragraphs — it has no punctuation glyphs
  • Use Plus Jakarta Sans for hero headings — that's Ritualist's job
  • Use Courier Prime for descriptions or body copy
  • Set any text below 14px — it's the system minimum
  • Mix more than 2 fonts on one surface (card, section, panel)
  • Use ink-3 for normal-sized body text (fails AA contrast)

Numbering rules
  • Font: var(--font-mono) Courier Prime, weight 700
  • Format: 0.01, 0.02, etc. — always with leading zero and decimal
  • Size: 64–80px, used as a background typographic element
  • Opacity: 8–15% of the text colour — never compete with actual content
  • Purpose: decorative wayfinding, not functional numbering. Screen readers should skip these (aria-hidden="true")
Editorial · 03

Text on image

Guidelines for placing typography directly on photography — the hallmark of the editorial magazine style.

a n s i k t
beauty
DO
  • Always use a gradient overlay behind text (min 0.5 opacity at text area)
  • Place text at the bottom or in corners — avoid covering the subject's face
  • Use Linen or Plaster for text colour on dark overlays
  • Keep text minimal — heading + one line max
  • Test contrast: text must meet 4.5:1 against the darkest part of the overlay
DON'T
  • Place text directly on an image without an overlay
  • Use body copy or paragraphs on images — save that for solid backgrounds
  • Use dark text on light areas of an image (unreliable contrast)
  • Cover the subject's eyes or key focal point with text
  • Use more than 2 lines of text on any image overlay

Imagery · 01

Photography guidelines

Derived from existing flipbook imagery and salon aesthetic. All photography should feel warm, natural, and considered.

Lighting

Warm, natural, side-lit. Golden hour quality. Avoid harsh flash or cool fluorescent lighting.

Colour Grading

Warm earth tones, slightly desaturated. Avoid high-saturation or cool/blue colour grading.

Skin

Natural texture, dewy, minimal retouching. Freckles and pores are welcome. No heavy airbrushing.

Composition

Close crops, eye-level or slightly above. Generous negative space in warm neutrals.

Environment

Bare plaster, natural materials, dried flowers, exposed wood, ceramic. No clinical or sterile settings.

Mood

Calm, considered, intimate. Not high-energy or overly glossy. Think editorial beauty, not commercial.

Accessibility · 01

AA compliance guide

All design decisions in this system are made with WCAG 2.1 AA compliance. This section provides practical guidance for maintaining accessibility.

Contrast Requirements

  • Normal text (below 24px / 18.67px bold): minimum 4.5:1 contrast ratio
  • Large text (24px+ or 18.67px+ bold): minimum 3:1 contrast ratio
  • UI components and graphical objects: minimum 3:1 contrast ratio
  • See the accessibility matrix for all approved pairings

Typography

  • System minimum: 14px — no text in the system may be set below this
  • Body copy: 16px recommended minimum for comfortable reading
  • Line-height: minimum 1.4 for body text, 1.6 recommended
  • Paragraph width: ~70 characters maximum for readability

Interactive Elements

  • Touch targets: minimum 44 × 44px on all interactive elements
  • Focus states: 2px solid var(--color-accent) outline with 2px offset
  • Focus must be visible on all interactive elements — never remove outline without an alternative
  • Keyboard navigation must reach all interactive elements in logical order

Colour Usage

  • Never use colour as the sole indicator of state or meaning
  • Always pair colour with text, icon, or pattern (e.g. error = red tint + red text + descriptive message)
  • The pregnancy-safe column in the treatment table uses colour and text ("Yes"/"No")

Images

  • All images must have descriptive alt text
  • Decorative images use alt=""
  • Complex images (charts, diagrams) need long descriptions via aria-describedby

Motion

  • Respect prefers-reduced-motion media query
  • No auto-playing animations
  • Any motion should be subtle and purposeful — matching the brand's calm aesthetic
